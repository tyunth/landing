# Math Landing

**Math Landing** - это веб-приложение для изучения математики, объединяющее библиотеку учебников и интерактивные задачи для учеников 5-11 классов.

## ✨ Особенности

### 📚 Учебники
- **Большая библиотека**: Собранные лучшие учебники по математике для всех классов
- **PDF просмотрщик**: Просмотр PDF файлов прямо в браузере без скачивания
- **Интеллектуальная фильтрация**: Поиск по классу, предмету, категории и ключевым словам
- **Скачивание**: Возможность скачивания учебников для офлайн использования

### 📐 Задачи на готовых чертежах
- **Интерактивные задачи**: Задачи с визуальными чертежами для лучшего понимания
- **Скрытые ответы**: Система проверки ответов с возможностью подсказок
- **Категоризация**: Задачи по классам и темам (геометрия, алгебра, тригонометрия)
- **Добавление задач**: Возможность добавлять новые задачи с изображениями

### 🎮 Мини-игры
- **Математический тренажер**: Решение примеров на время
- **Геометрические головоломки**: Определение фигур по описанию
- **Счет на скорость**: Тренировка устного счета
- **Система очков**: Подсчет баллов и рейтинговая система

### 📱 Адаптивный дизайн
- **Универсальная поддержка**: Оптимальное отображение на всех устройствах
- **Гибкая верстка**: Современные технологии CSS Grid и Flexbox
- **Быстрая загрузка**: Оптимизированные изображения и код
- **Доступность**: Поддержка screen readers и клавиатурного управления

## 📖 Документация

### API Endpoints

#### Учебники
- `GET /api/books` - Получить список учебников
- `POST /api/books` - Создать новый учебник
- `DELETE /api/books/:id` - Удалить учебник
- `POST /api/upload/pdf` - Загрузить PDF файл
- `POST /upload` - Простая загрузка файлов (для совместимости)

#### Задачи
- `GET /api/tasks` - Получить список задач
- `POST /api/tasks` - Создать новую задачу
- `DELETE /api/tasks/:id` - Удалить задачу
- `POST /api/upload/image` - Загрузить изображение

#### Фильтрация
- **Учебники**: По классу (5-11), предмету (math, algebra, geometry), категории (basic, additional, recommended), поиску
- **Задачи**: По классу (5-11), теме (geometry, algebra, trigonometry, stereometry), поиску
