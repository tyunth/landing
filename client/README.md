# Math Landing Client

Клиентская часть приложения Math Landing - веб-интерфейс для просмотра учебников и решения задач по математике.

## Технологии

- **HTML5** - Структура страницы
- **CSS3** - Стилизация и анимации
- **JavaScript (ES6+)** - Логика приложения
- **Font Awesome** - Иконки
- **Google Fonts** - Шрифты
- **PDF.js** - Просмотр PDF файлов

## Структура проекта

```
client/
├── index.html              # Главная страница
├── css/
│   ├── style.css          # Основные стили
│   └── responsive.css     # Адаптивные стили
├── js/
│   ├── main.js           # Главный JavaScript файл
│   ├── upload.js         # Система загрузки файлов
│   ├── pdf.js            # PDF просмотрщик
│   └── games.js          # Мини-игры
└── README.md             # Документация
```

### 🎮 Мини-игры
- Математический тренажер
- Геометрические головоломки

## Установка и запуск

1. Склонируйте репозиторий:
```bash
git clone <repository-url>
cd client
```

2. Запустите локальный сервер (например, через Live Server в VS Code):
- Откройте папку в VS Code
- Установите расширение "Live Server"
- Нажмите "Go Live" в правом нижнем углу

3. Или используйте Python:
```bash
python -m http.server 8000
```

4. Или используйте Node.js:
```bash
npx serve -s . -l 8000
```

## API Интеграция

Клиентская часть интегрирована с серверной частью через REST API:

- **Базовый URL**: `http://localhost:1000/api`
- **Учебники**: `/api/books`
- **Задачи**: `/api/tasks`
- **Загрузка файлов**: `/api/upload/pdf`, `/api/upload/image`
