# Math Landing - Инструкция по использованию

## 🚀 Быстрый старт

### Минимальные требования
- **Node.js** v14+
- **PostgreSQL** v12+
- **npm** или **yarn**
- Современный веб-браузер (Chrome 80+, Firefox 75+, Safari 13+, Edge 80+)

### Установка и запуск

#### Способ 1: Полная установка (рекомендуется)
1. Склонируйте репозиторий:
   ```bash
   git clone <repository-url>
   cd math-landing
   ```

2. Запустите автоматическую установку:
   ```bash
   run.bat
   ```
   Этот скрипт автоматически:
   - Установит все зависимости
   - Настроит базу данных
   - Запустит сервер
   - Откроет приложение в браузере

#### Способ 2: Пошаговая установка
1. **Установите зависимости:**
   ```bash
   install.bat
   ```

2. **Настройте базу данных:**
   ```bash
   setup-db.bat
   ```
   Вам нужно будет ввести:
   - Имя пользователя PostgreSQL
   - Пароль (если требуется)
   - Имя базы данных (по умолчанию: math_landing)

3. **Запустите сервер:**
   ```bash
   start.bat
   ```

4. **Откройте приложение:**
   Перейдите в браузере по адресу: http://localhost:1000

## 📚 Возможности приложения

### Учебники
- **Просмотр PDF**: Просматривайте учебники прямо в браузере
- **Фильтрация**: Ищите учебники по классу, предмету и категории
- **Поиск**: Находите учебники по названию, автору или описанию
- **Скачивание**: Скачивайте учебники для офлайн использования

### Задачи на готовых чертежах
- **Решение задач**: Решайте задачи с визуальными чертежами
- **Проверка ответов**: Система скрытых ответов для самопроверки
- **Фильтрация**: Ищите задачи по классу и теме
- **Добавление задач**: Возможность добавлять новые задачи

### Мини-игры
- **Математический тренажер**: Решайте примеры на время
- **Геометрические головоломки**: Определяйте фигуры по описанию
- **Счет на скорость**: Тренируйте устный счет
- **Система очков**: Зарабатывайте баллы за правильные ответы

### Контакты
- **Обратная связь**: Оставляйте сообщения и отзывы
- **Контактная информация**: Email, телефон, адрес

## 🔧 Администрирование

### Добавление учебников
1. Перейдите в раздел "Загрузка"
2. Перетащите PDF файл в область загрузки или выберите файл
3. Заполните информацию о учебнике (название, автор, класс и т.д.)
4. Нажмите "Сохранить"

### Добавление задач
1. Перейдите в раздел "Задачи"
2. Нажмите "Добавить задачу"
3. Заполните форму:
   - Класс
   - Тема
   - Описание задачи
   - Ответ
   - Изображение (по желанию)
4. Нажмите "Добавить задачу"

### Управление файлами
- **PDF файлы** сохраняются в `server/uploads/pdf/`
- **Изображения** сохраняются в `server/uploads/images/`
- **База данных** содержит метаданные о файлах

│       └── games.js       # Мини-игры
├── server/                 # Серверная часть (бэкенд)

├── README.md              # Основная документация

### Конфигурация сервера
Файл `.env` в папке `server/` содержит настройки:
```env
PORT=1000                    # Порт сервера
DB_HOST=localhost           # Хост базы данных
DB_PORT=5432               # Порт PostgreSQL
DB_NAME=math_landing       # Имя базы данных
DB_USER=your_username      # Пользователь базы данных
DB_PASSWORD=your_password  # Пароль базы данных
DB_SSL=false               # SSL подключение
MAX_FILE_SIZE=50000000     # Максимальный размер файла (50MB)
CORS_ORIGIN=http://localhost:1000  # Разрешенные источники
```

### API Endpoints
- `GET /api/books` - Получить список учебников
- `POST /api/books` - Создать новый учебник
- `DELETE /api/books/:id` - Удалить учебник
- `POST /api/upload/pdf` - Загрузить PDF файл
- `GET /api/tasks` - Получить список задач
- `POST /api/tasks` - Создать новую задачу
- `DELETE /api/tasks/:id` - Удалить задачу
- `POST /api/upload/image` - Загрузить изображение

## 🐛 Устранение неполадок

### Проблемы с запуском сервера
1. **Проверьте Node.js:**
   ```bash
   node --version
   npm --version
   ```

2. **Проверьте зависимости:**
   ```bash
   cd server
   npm install
   ```

3. **Проверьте порт:**
   Убедитесь, что порт 1000 не занят другим приложением

### Проблемы с базой данных
1. **Проверьте PostgreSQL:**
   ```bash
   psql --version
   ```

2. **Проверьте подключение:**
   Убедитесь, что PostgreSQL сервер запущен

3. **Проверьте учетные данные:**
   Проверьте правильность логина и пароля в `.env` файле

### Проблемы с клиентом
1. **Проверьте браузер:**
   Используйте современный браузер с поддержкой ES6+

2. **Проверьте консоль:**
   Откройте Developer Tools и проверьте ошибки в консоли

3. **Проверьте CORS:**
   Убедитесь, что клиент и сервер работают на правильных портах

### Проблемы с загрузкой файлов
1. **Проверьте папки:**
   Убедитесь, что папки `server/uploads/pdf` и `server/uploads/images` существуют

2. **Проверьте права:**
   Убедитесь, что у сервера есть права на запись в эти папки

3. **Проверьте размер файлов:**
   Файлы не должны превышать 50MB для PDF и 10MB для изображений
